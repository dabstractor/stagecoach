## Why

- **The system prompt is the style-learning + output-contract surface.** PRD FR13/FR14: the prompt carries the role ("commit message generator"), the output contract, the essence-not-filenames instruction, the style examples with the explicit anti-reuse prohibition, the conditional multi-line rule, and the subject-length target. `BuildSystemPrompt` is the single function that assembles all of this from the S1-provided `examples`/`hasMultiline` plus the caller-supplied `newRepo`/`target` scalars.
- **RAW output, not JSON — the v1 design call (§17.4, reference D1).** The reference `commit-pi` used `{"commit_message":"…"}` parsed with sed, which constrained the model (no double quotes) and was fragile. Go's v1 default is RAW ("output only the message"). So the prompt MUST emit the raw-output contract line and MUST NOT mention JSON anywhere. JSON remains an optional per-provider manifest setting (`output="json"`), handled by the parse layer (M2) — NOT by this prompt. This task is the place that anti-JSON invariant is enforced.
- **Decoupled from git AND config (plan_overview: M4 is decoupled; builders take scalar settings).** The prompt package must not import `internal/git` or `internal/config`. The builder takes `newRepo bool` (the caller — M6 generate — derives it from `CommitCount<=1`, the same gate S1's FetchExamples uses) and `target int` (the caller — M6 — reads from config `subject_target_chars`, default 50, in M5). `DefaultSubjectTargetChars = 50` is the canonical default the generate layer passes before config wiring, mirroring S1's `DefaultExampleCount = 20`.
- **Canonical strings committed as Go constants (PRD Appendix A).** Appendix A mandates the §17.1/§17.2 templates be committed verbatim as named Go string constants so the exact wording is reviewable and stable, with only diff/examples/rejection-list interpolated at runtime (the `target` char count interpolates the same way since it is a scalar parameter).

### Implementation Patterns & Key Details
```go
func BuildSystemPrompt(examples []string, hasMultiline bool, newRepo bool, target int) string {
    if newRepo {
        return strings.Join([]string{
            systemRole,
            newRepoRawContract, 
            essenceInstruction,
            fmt.Sprintf("Target ~%d characters (~7 words). Format: type(scope): description", target),
        }, "\n\n")
    }
    sections := []string{
        systemRole,
        matureRawContract, 
        essenceInstruction,
        renderExamples(examples),                 
        antiReuseClause,
    }
    if hasMultiline {
        sections = append(sections, multilineRuleMulti)
    } else {
        sections = append(sections, multilineRuleSingle)
    }
    sections = append(sections, fmt.Sprintf("Target ~%d characters for the subject line.", target))
    return strings.Join(sections, "\n\n")
}
func renderExamples(examples []string) string {
    var b strings.Builder
    b.WriteString(examplesHeader)
    b.WriteByte('\n')
    for _, ex := range examples {
        b.WriteString(exampleSeparator)
        b.WriteByte('\n')
        b.WriteString(ex)
        b.WriteByte('\n')
    }
    b.WriteString(examplesFooter)
    return b.String()
}
```
