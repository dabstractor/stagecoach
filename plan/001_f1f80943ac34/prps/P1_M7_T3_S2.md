## User Persona

**Target User**: a new stagehand user running first-time onboarding.
**Use Case**: discover where stagehand expects its config file, and scaffold a fully-documented, commented template at that path to edit by uncommenting lines.
**User Journey**: `stagehand config path` → see `$XDG_CONFIG_HOME/stagehand/config.toml` (or `~/.config/...`) → `stagehand config init` → a commented template is written, every key documented inline → optionally add `./.stagehand.toml` to `.gitignore` → edit the file by uncommenting the lines they want.
**Pain Points Addressed**: "where does my config live?" (XDG is non-obvious) and "what keys exist and what are the defaults?" (the generated file IS the reference — Mode A).

## Why

- **Single source of truth for the config-file reference (Mode A)**: the generated commented file documents every key inline, so the reference can never drift from the code that reads it.
- **Wires already-built dependency**: consumes `config.GlobalConfigPath()` (P1.M5.T2.S1, DONE) — itself documented as "the single source of truth for both readGlobalFile and the CLI `config path` / `config init` subcommands (P1.M7.T3.S2)". This task is pure CLI glue + the template string; no new domain logic.

## What

### `stagehand config path` (FR38)
- Resolve `p, err := config.GlobalConfigPath()`; on non-nil err return it (exit 1). Print `p` to stdout (exit 0). `cobra.NoArgs`.

### `stagehand config init` (FR38)
- Boolean local flag `--force` (default false) on the `init` command (NOT a persistent flag — do not touch rootCmd persistent flags, owned by P1.M7.T2.S1).
- Resolve `p, err := config.GlobalConfigPath()`; on non-nil err return it (exit 1).
- If a file already exists at `p` and `--force` is false → return `fmt.Errorf("config: %s already exists (rerun with --force to overwrite)", p)` (exit 1, NO write).
- Otherwise: `os.MkdirAll(filepath.Dir(p), 0o755)`; `os.WriteFile(p, []byte(config.ExampleConfig()), 0o644)`; print `Wrote commented example config to %s` to stdout.
- Then `offerGitignore(out, in, cwd)`: only if `cwd/.git` exists. Print `Add ./.stagehand.toml to your local .gitignore? [y/N] `; read one trimmed line from `in`; if it is `y`/`yes` (case-insensitive), append a line `.stagehand.toml` to `cwd/.gitignore` (creating the file if absent, with a trailing newline) and print confirmation; any other answer (including empty/EOF) → no-op. `cobra.NoArgs`.

### Implementation Patterns & Key Details
```go
func writeExampleConfig(out io.Writer, path string, force bool) error {
    if !force {
        if _, err := os.Stat(path); err == nil {
            return fmt.Errorf("config: %s already exists (rerun with --force to overwrite)", path)
        }
    }
    if err := os.MkdirAll(filepath.Dir(path), 0o755); err != nil { return err }
    if err := os.WriteFile(path, []byte(config.ExampleConfig()), 0o644); err != nil { return err }
    fmt.Fprintf(out, "Wrote commented example config to %s\n", path)
    return nil
}
func offerGitignore(out io.Writer, in io.Reader, cwd string) error {
    if _, err := os.Stat(filepath.Join(cwd, ".git")); err != nil { return nil } 
    fmt.Fprint(out, "Add ./.stagehand.toml to your local .gitignore? [y/N] ")
    sc := bufio.NewScanner(in)
    if !sc.Scan() { return nil }                                              
    if !isYes(sc.Text()) { return nil }
    gi := filepath.Join(cwd, ".gitignore")
    f, err := os.OpenFile(gi, os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0o644)
    if err != nil { return err }
    defer f.Close()
    if _, err := fmt.Fprintln(f, ".stagehand.toml"); err != nil { return err }
    fmt.Fprintln(out, "Added ./.stagehand.toml to .gitignore")
    return nil
}

```
