# PRP — P1.M1.T1.S1: go.mod, cmd/stagehand/main.go cobra stub, Makefile, .gitignore

## Goal

**Feature Goal**: Establish the foundational Go module `github.com/dustin/stagehand`, a runnable cobra CLI stub whose `--version` flag reports the build version, a Makefile with the canonical build/test/lint/coverage targets wired to version-injecting ldflags, and repo-hygiene files. This is the entry point of the build DAG — every downstream package (internal/ui, internal/provider, …) imports this module path.

**Deliverable**: A working repo with `go.mod` + `go.sum`, `cmd/stagehand/main.go`, `Makefile`, `.gitignore` (+ optional `.editorconfig`) such that `make build` produces `./bin/stagehand` and `./bin/stagehand --version` prints the version and exits 0.

**Success Definition**: `make build` → `./bin/stagehand`; `./bin/stagehand --version` → `stagehand version <git-describe>` on stdout, exit 0 (NOT the `dev` default — proves ldflags injection); bare `./bin/stagehand` → prints help, exit 0. All Makefile targets present. Module importable by downstream packages.

## Why

- This is the **root of the acyclic dependency DAG** (plan_overview.md M1). Nothing else can be built until the module + a compiling binary exist.
- Establishes the module path `github.com/dustin/stagehand` that every `internal/*` and `pkg/stagehand` package imports, and the `go install github.com/dustin/stagehand/cmd/stagehand@latest` distribution path (PRD §21.3).
- Pins the two runtime deps (cobra, go-toml/v2) so M2/M5 don't re-negotiate versions.
- Keeps a clean **v2-readiness / wiring seam**: the root command has NO `Run` yet — the default action (`maybeAutoStage(); CommitStaged()`) is wired in M7.T2 (decisions.md §1, §6). This task must not entangle staging/commit logic.

## What (user-visible behavior & technical requirements)

1. `make build` compiles `./cmd/stagehand` to `./bin/stagehand`, injecting the version via `-ldflags "-X main.version=$(VERSION)"` where `VERSION := $(shell git describe --tags --always --dirty)`.
2. `./bin/stagehand --version` prints `stagehand version <VERSION>` to **stdout** and exits 0.
3. `./bin/stagehand` (no args) prints cobra help to stdout and exits 0 (no `Run`).
4. `go.mod` module = `github.com/dustin/stagehand`; deps cobra v1.10.2 + pelletier/go-toml/v2 v2.4.2 present.
5. Makefile targets: `build`, `test`, `coverage`, `lint`, `vet`, `fmt`, `clean`.
6. `.gitignore` ignores `bin/`, `*.out`, `coverage.out`, `dist/`.
